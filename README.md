vimrc
------------
My vimrc setting for vim editor
[![Screenshot](vimrc_screenshot.jpg)](https://raw.githubusercontent.com/z-sean-huang/vimrc/master/vimrc_screenshot.jpg)


Installation
------------
    cd /tmp
    git clone https://github.com/z-sean-huang/vimrc.git
    cd vimrc
    sh install.sh
